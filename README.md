# Website for Makers League e.V.

## Tools used

- [TypeScript](https://www.typescriptlang.org)
- [React](https://reactjs.org)
- [Next.js](https://nextjs.org)
- [Tailwind CSS](https://tailwindcss.com)

## How to run locally

- Add `.env` file with Easyverin API token
- Run `npm install`
- Run `npm run dev`
